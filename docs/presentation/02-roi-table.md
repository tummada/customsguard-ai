# VOLLOS CustomsGuard — ตาราง ROI (Return on Investment)

---

## ความคุ้มค่าที่วัดได้

| หัวข้อความคุ้มค่า | รายละเอียดการประหยัด | ผลลัพธ์ที่ได้ |
|---|---|---|
| **FTA Optimization** | Savings = มูลค่า x (อัตราปกติ% - อัตรา FTA%) | ประหยัดภาษีให้ลูกค้า **หลักแสน/shipment** |
| **Amendment Cost** | ลดโอกาสคีย์ผิดด้วย AI Scan + ล็อคเรทกรมศุลฯ | ลดค่าปรับ/ค่าแก้ใบขน **หลักหมื่น/เดือน** |
| **Time Efficiency** | จากคีย์มือ 30 นาที → ตรวจ Draft 2 นาที | รับงานได้เพิ่มขึ้น **5-10 เท่า** |
| **Liability Protection** | Audit Trail เก็บ Log เอกสารที่อ้างอิง | **ปลอดภัย** เวลาโดนตรวจย้อนหลัง |

---

## ตัวอย่างการคำนวณ FTA Savings รายปี

### สมมติ: นำเข้าจากญี่ปุ่น เดือนละ 4 shipment มูลค่าเฉลี่ย 1,000,000 บาท/shipment

| รายการ | คำนวณ | ผลลัพธ์ |
|---|---|---|
| อัตราภาษีปกติ (MFN) | 10% x 1,000,000 | 100,000 บาท/shipment |
| อัตราภาษี JTEPA | 3.5% x 1,000,000 | 35,000 บาท/shipment |
| **ประหยัดต่อ shipment** | 100,000 - 35,000 | **65,000 บาท** |
| **ประหยัดต่อเดือน** | 65,000 x 4 shipments | **260,000 บาท** |
| **ประหยัดต่อปี** | 260,000 x 12 เดือน | **3,120,000 บาท** |

---

## ตัวอย่างการลดค่า Amendment

### สมมติ: ปัจจุบันแก้ใบขน 5 ครั้ง/เดือน ค่าแก้ครั้งละ 2,000-5,000 บาท

| รายการ | ก่อนใช้ระบบ | หลังใช้ระบบ |
|---|---|---|
| จำนวนครั้งที่แก้ใบขน/เดือน | 5 ครั้ง | 1 ครั้ง (ลด 80%) |
| ค่า Amendment เฉลี่ย | 3,500 บาท/ครั้ง | 3,500 บาท/ครั้ง |
| **ค่า Amendment ต่อเดือน** | **17,500 บาท** | **3,500 บาท** |
| **ประหยัดต่อเดือน** | | **14,000 บาท** |
| **ประหยัดต่อปี** | | **168,000 บาท** |
