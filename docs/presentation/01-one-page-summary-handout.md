# VOLLOS CustomsGuard — ผู้ช่วย AI สำหรับงานพิธีการศุลกากร

---

## เปรียบเทียบ: วิธีเดิม vs ด้วย VOLLOS CustomsGuard

| ขั้นตอนงาน | วิธีเดิม (Manual) | ด้วย VOLLOS CustomsGuard |
|---|---|---|
| **อ่านข้อมูลจาก PDF** | พิมพ์มือ (เสี่ยงพิมพ์ผิด) | AI Scan + OCR ดึงข้อมูลอัตโนมัติ |
| **ค้นหา HS Code** | เปิดเล่มพิกัด / ค้น Keyword | AI Semantic Search (เข้าใจบริบท) |
| **ตรวจสอบ FTA** | เช็ค Manual ทีละฉบับ | Auto-Alert + คำนวณเงินที่ประหยัดได้ |
| **ตรวจสอบกฎหมาย** | ค้นประกาศกรมฯ ย้อนหลังเอง | RAG Chat (ถามได้ — ตอบมีอ้างอิง) |
| **เรทแลกเปลี่ยน** | เช็ค Manual ทุกสัปดาห์ | ดึงเรทประกาศกรมศุลฯ อัตโนมัติ |
| **ความเสี่ยง Post-Audit** | ต้องจำเอกสารอ้างอิงเอง | Audit Trail (Log เก็บหลักฐานอ้างอิง) |
| **ระบบค้าง/งานหาย** | คีย์ใหม่ตั้งแต่ต้น | Auto-Save Draft ในเครื่อง |

## ขั้นตอนการทำงาน 4 Steps

```
  Step 1               Step 2                Step 3               Step 4
┌──────────┐       ┌──────────────┐      ┌──────────────┐     ┌──────────────┐
│  ลาก PDF │──────>│  AI เสนอพิกัด │─────>│  พี่ตรวจ +   │────>│  กดปุ่มเดียว │
│   วาง    │       │  + แจ้ง FTA   │      │  ยืนยัน      │     │  ลงหน้าเว็บ  │
│          │       │  Savings     │      │  (แก้ไขได้)   │     │ หรือ Export  │
└──────────┘       └──────────────┘      └──────────────┘     │ XML ไปใช้ต่อ│
                                                               └──────────────┘
  AI Scan +          AI ทำ Draft            Human-in-            Output
  OCR อัตโนมัติ       + คำนวณเงิน            the-loop             พร้อมใช้งาน
```

**จุดขาย:** แค่ 4 ขั้นตอน จากเดิมที่ต้องเปิดหลายระบบ → ทำในที่เดียว

## สูตรคำนวณส่วนต่างภาษี FTA

> **Savings = มูลค่าสินค้า x (อัตราปกติ% - อัตรา FTA%)**

### ตัวอย่างจริง:

| ประเทศต้นทาง | มูลค่า (บาท) | อัตราปกติ | FTA | อัตรา FTA | ประหยัดได้ |
|---|---|---|---|---|---|
| ญี่ปุ่น | 1,000,000 | 10% | JTEPA | 3.5% | **65,000 บาท** |
| จีน | 500,000 | 15% | ACFTA | 5% | **50,000 บาท** |
| อาเซียน | 800,000 | 12% | ATIGA | 0% | **96,000 บาท** |

**ต่อ 1 shipment — ยิ่งนำเข้าบ่อย ยิ่งประหยัดมาก**
