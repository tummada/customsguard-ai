## 10 สถานการณ์จริง + วิธีสั่ง

---

**① เปิด landing page ใหม่ — ต้องการภาพ Hero**

*สถานการณ์: กำลังสร้างหน้าแรกของ vollos.ai อยากได้ภาพ cinematic เปิดหน้าเว็บ*

```
อยากได้ image prompt สำหรับ hero section หน้าแรกของ VOLLOS 
ใช้ Midjourney อัตราส่วน 16:9
```

---

**② โพสต์ LinkedIn หาลูกค้า B2B**

*สถานการณ์: อยากโพสต์ LinkedIn เจาะกลุ่ม CFO และเจ้าของบริษัทชิปปิ้ง*

```
ช่วยสร้าง image prompt + copywriting สำหรับโพสต์ LinkedIn 
เป้าหมายคือ CFO บริษัทนำเข้า อยากให้เน้นความเสี่ยงภาษีย้อนหลัง
```

---

**③ มีตัวเลข ROI จริง — อยากทำ visual โชว์ลูกค้า**

*สถานการณ์: เก็บ data มาได้ว่า VOLLOS ช่วยลดเวลาจาก 3 ชั่วโมงเหลือ 2 นาที อยากทำภาพโชว์ตัวเลขนี้*

```
ทำ visual สำหรับโชว์ ROI ของ VOLLOS 
ตัวเลขจริงคือ: ลดเวลาจาก 3 ชั่วโมง เหลือ 2 นาทีต่อใบขน
ใช้สำหรับ sales deck ส่ง prospect
```

---

**④ ทำโฆษณา Facebook รอบแรก**

*สถานการณ์: จะยิง Facebook Ad ครั้งแรก งบจำกัด อยากได้ concept ที่หยุด scroll ได้*

```
ต้องการ video concept 15 วินาที สำหรับ Facebook Ad
กลุ่มเป้าหมาย: พนักงานคีย์ใบขนและเจ้าของบริษัทชิปปิ้ง
เน้นความเร็ว ต้องหยุด scroll ได้ใน 2 วินาทีแรก
```

---

**⑤ ทำ feature card สำหรับหน้า pricing/features**

*สถานการณ์: กำลังออกแบบหน้า features ของเว็บ อยากได้ icon illustration แต่ละ feature*

```
ช่วยทำ image prompt สำหรับ feature card ของ Magic Fill
ใช้ใน website section แสดง features ทั้งหมด
อัตราส่วน 1:1
```

---

**⑥ คอนเทนต์ TikTok สำหรับ awareness**

*สถานการณ์: อยากลอง TikTok เจาะคนรุ่นใหม่ที่เพิ่งเริ่มทำธุรกิจนำเข้า*

```
ทำ video storyboard สำหรับ TikTok ความยาว 30 วินาที
เรื่อง: คนทั่วไปไม่รู้ว่าตัวเองเสียเงินภาษีเกินจริงอยู่เท่าไหร่
อยากให้มีไทยซับไตเติ้ลด้วย
```

---

**⑦ เตรียม pitch deck หา investor**

*สถานการณ์: กำลังเตรียม deck สำหรับ pitch กับ VC อยากได้ visual ที่ดูจริงจังและน่าเชื่อถือ*

```
ต้องการ visual concept สำหรับ investor pitch deck ของ VOLLOS
สไลด์ที่ต้องการ: Problem slide และ Solution slide
ผู้ดู: VC ที่ไม่คุ้นเคยกับ customs workflow
```

---

**⑧ ทำ email campaign หาลูกค้าใหม่**

*สถานการณ์: จะส่ง cold email ไปหา shipping company 50 บริษัท อยากได้ header image ที่โปรฯ*

```
ทำ image prompt สำหรับ email header
ส่งให้ผู้จัดการแผนกชิปปิ้ง เน้นความน่าเชื่อถือและ professional
ขนาด 600x200px landscape
```

---

**⑨ ทำ content series เดือนนี้ทั้งเดือน**

*สถานการณ์: อยากวางแผน content ล่วงหน้า 1 เดือน ไม่ต้องมานึกเองทุกวัน*

```
ช่วยวาง content calendar สำหรับ VOLLOS เดือนนี้
ช่องทาง: LinkedIn + Facebook + Instagram
focus feature ของเดือนนี้คือ Magic Fill และ Audit Traffic Light
```

---

**⑩ มีลูกค้าให้ feedback มา — อยากทำ testimonial visual**

*สถานการณ์: ลูกค้าพูดว่า "ก่อนใช้ VOLLOS ต้องนั่งคีย์ 4 ชั่วโมงต่อวัน ตอนนี้เหลือ 15 นาที" อยากทำภาพ quote สวยๆ*

```
ทำ testimonial visual quote card
quote: "จาก 4 ชั่วโมง เหลือ 15 นาที ต่อวัน"
ใช้โพสต์ Instagram อัตราส่วน 4:5
อยากได้ทั้ง image prompt และ copy layout แนะนำ
```

## หลักการเลือกสั่ง

สั้นๆ คือบอก **3 อย่าง** ก็พอครับ:

```
[อยากได้อะไร] + [ใช้ที่ไหน/platform] + [กลุ่มเป้าหมายหรือตัวเลขจริง ถ้ามี]
```

ยิ่งมีตัวเลขจริงหรือ context ธุรกิจให้ — output จะยิ่งแม่นและ on-brand มากขึ้นเท่านั้นครับ
