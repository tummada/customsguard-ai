## เอกสาร Invoice ที่ควรเตรียมสำหรับทดสอบ

1. **Invoice กุ้งจีน (ภาษาอังกฤษ)** -- ทดสอบ HS code 0306 + ACFTA + LPI
2. **Invoice ข้าว (ภาษาไทย)** -- ทดสอบ OCR ภาษาไทย + ตัวเลขไทย
3. **Invoice อิเล็กทรอนิกส์ยุโรป (EUR)** -- ทดสอบสกุลเงิน EUR + EU number format
4. **Invoice หลายรายการ (5+ items)** -- ทดสอบตาราง + Traffic Light หลายตัว
5. **Invoice PDF หลายหน้า (3-5 หน้า)** -- ทดสอบ multi-page
6. **Invoice PDF ขนาดใหญ่ (5-10 MB)** -- ทดสอบ performance
7. **ไฟล์ที่ไม่ใช่ invoice** -- เช่น PDF รายงาน, รูป JPG
8. **Invoice สินค้า excise** -- เช่น เครื่องดื่มแอลกอฮอล์ สำหรับทดสอบสรรพสามิต
